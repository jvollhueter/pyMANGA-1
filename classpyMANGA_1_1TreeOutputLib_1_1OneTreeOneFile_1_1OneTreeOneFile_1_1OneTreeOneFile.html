pyMANGA.TreeOutputLib.OneTreeOneFile.OneTreeOneFile.OneTreeOneFile Class Reference

Detailed Description

Output class.

This class creates one file per tree at a defined location. A line containing time, position, desired geometric measures and desired parameters is written at every nth timestep.
